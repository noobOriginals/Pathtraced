# Pathtraced (reworked)

A CPU pathtracer made from scratch, with the goal of enforcing basic raytracing/pathtracing notions and implementing ALL the newest path-tracing tehniques and optimizations (in the long run).  
Project is marked as "reworked" (for now) because the main branch is still on the initial "Simple CPU Path Tracer", and all the work until now was just reworking the initial project in order to be compatible with the new (and heavy) system that will be implemented.
